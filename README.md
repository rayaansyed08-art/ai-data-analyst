# ai-data-analyst
AI Data Analyst using LLMs and RAG
This system uses Retrieval-Augmented Generation (RAG) to enhance accuracy.

- User queries are matched with dataset context
- Relevant data is retrieved using vector search (FAISS)
- The LLM generates responses based on grounded context
- Guardrails ensure safe and controlled outputs

This improves reliability compared to standard LLM responses.
