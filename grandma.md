Grandma Explanation

This project is like a smart assistant that helps understand data.

Imagine you have a file with numbers (like sales or customer feedback). Normally, you need a data analyst to understand it.

This system allows you to:

Upload your data 
Ask questions in simple English 
Get answers instantly
See charts and trends 

It works like a human expert but faster.

In simple words:

It reads data
It understands your question
It gives useful answers
It shows insights visually

So even someone with no technical knowledge can understand complex data easily.
